# World GDP Prediction Rate
# 📌 Overview

This project predicts the GDP (Gross Domestic Product) growth rate of countries using historical economic data and Machine Learning techniques. The goal is to analyze past trends and forecast future GDP growth rates.

# 🚀 FeaturesData Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- GDP Trend Visualization
- Machine Learning Model for Prediction
- Performance Evaluation

# 🛠️ Technologies Used
-Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

The dataset contains historical GDP data along with other economic indicators such as population, inflation, and unemployment rates.

# 🎯 Objective

To build a machine learning model that can predict GDP growth rates and help understand economic trends.
